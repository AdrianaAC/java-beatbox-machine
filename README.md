# Java Beatbox Machine

A small **Java desktop application** that generates rhythm patterns using **MIDI sequencing**.

The project was built while learning Java and experimenting with **Swing GUI development and MIDI sound playback**.

Users can activate different instruments on a grid to create simple beat patterns.

---

## Features

- Grid-based rhythm sequencer
- Play and stop beat patterns
- Adjust tempo
- MIDI-based sound generation
- Desktop GUI built with Java Swing

---

## Tech Stack

- Java
- Swing (GUI)
- MIDI sequencing

---

## How to Run

Compile the Java files:

```bash
javac BeatBox.java
```

Run the application:

```bash
java BeatBox
```

This will launch the desktop GUI application.
